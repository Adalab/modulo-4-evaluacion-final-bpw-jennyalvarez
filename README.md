API REST de Librería

URL BASE:

http://localhost:3000

ENDPOINTS:

1- Listar todos los libros
Método: GET
URL: /books o /

2- Crear un libro
Método: POST
URL: /books
Body: JSON

3- Actualizar un libro
Método: PUT
URL: /books/:id (se reemplazaría el :id por el idbooks del libro)
Body: JSON

4- Eliminar un libro
Método: DELETE
URL: /books/:id (se reemplazaría el :id por el idbooks del libro)
